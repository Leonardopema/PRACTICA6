¿Que es HTML?

El Lenguaje de Marcado de Hipertexto (HTML) es el código que se utiliza para estructurar y desplegar una página web y sus contenidos

¿Que es CSS?

Es el lenguaje de diseño grafico y crear la presentación de un documento estructurado escrito en lenguaje de marcado
